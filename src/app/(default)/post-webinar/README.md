# Post-Webinar Page

This page displays webinar content after the event has concluded.

## Structure

- **page.tsx** - Main page component
- **utils/webinar-hero-section/** - Hero banner with title and speaker
- **utils/webinar-content-section/** - Main content including summary, takeaways, experts, and download

## Features

1. **Hero Section**
   - Teal background with title
   - Banner image with speaker overlay
   - Responsive layout

2. **Content Section**
   - Summary with image
   - Key takeaways (purple gradient box)
   - Meet our experts (3 cards)
   - Download button
   - Image gallery

## Images Needed

Place these images in `/public/assets/webinar/`:
- banner.jpg
- speaker.jpg
- summary-image.jpg
- expert1.jpg, expert2.jpg, expert3.jpg
- gallery-1.jpg, gallery-2.jpg, gallery-3.jpg

## Mobile Responsive

All sections are fully responsive with mobile-first design.
