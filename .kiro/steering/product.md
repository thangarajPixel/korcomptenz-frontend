# Product Overview

korcomptenz-frontend is a corporate website built with modern web technologies. The application showcases company information, services, case studies, achievements, and provides features like job listings, news sections, and contact forms.

## Key Features

- Dynamic content sections (services, case studies, achievements, insights)
- Interactive UI components (sliders, galleries, sticky cards)
- Form handling (demo booking, newsletter, contact)
- Responsive design with dark mode support
- SEO-optimized with Next.js metadata
